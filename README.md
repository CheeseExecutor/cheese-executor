# 🧀 CheeseExecutor

## 📌 About
CheeseExecutor is a lightweight script execution environment designed for testing and learning Lua-based scripts in a simple and clean interface.

## ⚡ Features
- Fast and responsive script execution
- Built-in code editor
- Syntax highlighting
- Simple and clean UI
- Project workspace system
- Custom themes support

## 🎯 Purpose
This project is made for educational and development purposes only, helping users learn scripting in a safe and controlled environment.

## 🖥️ How to Use
1. Open the application
2. Paste or write your script in the editor
3. Click "Execute"
4. View output in the console

## 📁 Status
Early development / beta version

## ⚠️ Disclaimer
CheeseExecutor is intended for learning and testing purposes only. Misuse of this project is not supported.
